Key Features

Out-of-Order Execution

Full OoO engine with instruction window (ROB), register renaming, dependency tracking, and multi-issue scheduling.

Multi-Level Cache

Configurable L1/L2/L3 cache hierarchy with LRU replacement, DDR memory controller with bank-level timing.

CHI Coherence

AMBA CHI protocol with RN-F, HN-F, and SN-F node models, directory-based coherence.

ELF Execution

Direct ELF binary execution via Capstone v5 disassembly (~500 ARM64 mnemonics) with functional pre-simulation.

Konata Visualization

Pipeline timeline viewer with per-instruction stage tracking, color-coded by type, zoomable and searchable.

Multi-Instance Sim

Run multiple simulations in parallel with independent configurations, ideal for design space exploration.

Pipeline Overview

Fetch → Decode → Rename → Dispatch → Issue → Execute → Memory → Writeback → Commit
