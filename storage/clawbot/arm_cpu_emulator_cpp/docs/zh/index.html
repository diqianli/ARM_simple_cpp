核心特性

乱序执行

完整的 OoO Engine，包含指令窗口（ROB）、寄存器重命名、依赖追踪和多发射调度。

多级 Cache

可配置的 L1/L2/L3 Cache 层次结构，支持 LRU 替换策略与 DDR 内存控制器的 bank 级时序建模。

CHI 一致性协议

AMBA CHI 协议，包含 RN-F、HN-F 和 SN-F 节点模型，基于目录的一致性机制。

ELF 直接执行

通过 Capstone v5 反汇编直接执行 ELF 二进制文件（约 500 条 ARM64 助记符），并支持功能预仿真。

Konata 可视化

Pipeline 时间线查看器，支持逐指令阶段追踪、按类型着色、缩放与搜索功能。

多实例仿真

以独立配置并行运行多个仿真实例，适用于设计空间探索（DSE）场景。

Pipeline 概览

Fetch → Decode → Rename → Dispatch → Issue → Execute → Memory → Writeback → Commit
