基准测试

用于评估仿真器性能和微架构行为各个方面的基准测试程序套件。

项目在 benchmarks/ 目录下包含一组基准测试程序。每个基准测试包含 C 源文件（_source.c 或 .c）、编译后的 ARM64 二进制文件（_aarch64）以及 GEM5 SE 变体（_gem5_se）。

基准测试列表

1. Dhrystone

benchmarks/dhrystone/

经典的整数基准测试，用于通用性能评估。

  • 测试内容：整数算术、控制流、字符串处理
  • 适用场景：基线 IPC 测量
./build/arm_cpu_sim -f elf benchmarks/dhrystone/dhrystone_aarch64

2. Fibonacci

benchmarks/fibonacci/

递归 Fibonacci 计算。

  • 测试内容：分支预测、数据依赖、调用/返回开销
  • 适用场景：评估分支预测器准确率
./build/arm_cpu_sim -f elf benchmarks/fibonacci/fibonacci_aarch64

3. MatMul 4x4

benchmarks/matmul/

4x4 矩阵乘法。

  • 测试内容：乘累加、内存访问模式
  • 适用场景：FP/SIMD 和内存子系统评估
./build/arm_cpu_sim -f elf benchmarks/matmul/matmul_aarch64

4. QuickSort

benchmarks/qsort/

对整数数组进行快速排序。

  • 测试内容：分支预测（分区操作）、递归调用
  • 适用场景：分支预测挑战、数据依赖行为
./build/arm_cpu_sim -f elf benchmarks/qsort/qsort_aarch64

5. LinkedList

benchmarks/linkedlist/

链表的遍历和操作。

  • 测试内容：不规则内存访问模式
  • 适用场景：Cache 性能评估、指针追踪
./build/arm_cpu_sim -f elf benchmarks/linkedlist/linkedlist_aarch64

6. MemcpyBench

benchmarks/memcpy_bench/

内存拷贝基准测试。

  • 测试内容：纯内存带宽和吞吐量
  • 适用场景：内存子系统压力测试
./build/arm_cpu_sim -f elf benchmarks/memcpy_bench/memcpy_bench_aarch64

运行全部基准测试

可以通过循环遍历所有基准测试：

for bench in benchmarks/*/*.c; do
    echo "=== $bench ==="
    # 编译并运行每个基准测试
done

对比结果

使用不同的预设来比较不同配置下的性能：

  • 对比预设：--preset default vs --preset high_performance
  • 对比 Cache 配置：--l1-size 32 vs --l1-size 128
  • 跨基准测试跟踪 IPC、Cache 命中率和分支预测准确率
分析跨基准测试趋势

使用相同配置运行每个基准测试并收集输出 JSON 文件。然后可以比较 IPC、Cache 缺失率和分支误预测率，了解不同的微架构特性如何影响每个工作负载。
